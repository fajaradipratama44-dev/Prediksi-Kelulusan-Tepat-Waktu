Sistem Prediksi Kelulusan Mahasiswa

Proyek End-to-End Machine Learning untuk memprediksi ketepatan waktu kelulusan mahasiswa angkatan 2022 menggunakan Random Forest Classifier. Proyek ini dilengkapi dengan Dashboard Interaktif menggunakan Streamlit.

Fitur Utama
1. Dashboard Monitoring: Visualisasi statistik mahasiswa.
2. System Early Warning: Deteksi otomatis mahasiswa yang berisiko terlambat.
3. Prediksi Mandiri: Input data akademik untuk memprediksi status kelulusan.
4. Daftar Mahasiswa Berisiko: Melihat mahasiswa yang tidak lulus tepat waktu.

Cara Menjalankan Aplikasi
1. Clone repositori ini.
2. Instal dependensi: `pip install -r requirements.txt`
3. Jalankan aplikasi: `streamlit run Dashboard_Kelulusan.py`
